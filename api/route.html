\Can\Server\Route API¶

class Route¶

Route class.

Can\Server\Route::__construct(string $uri, callable $handler, int $methods = Route::METHOD_GET)¶

Constructor. Instantiates new Server route.

Parameters:
  • $uri (string) – Static or dynamic URI path.
  • $handler (callable) – Request handler - any valid PHP callback
  • $methods (int) – HTTP request methods bitmask this route associated with, default Route::METHOD_GET.
Throws :
  • InvalidParametersException If invalid parameters are passed to the method.

The request handler must accepting Request instance as first parameter and associative arguments array as second parameter in which you will find values of the wildcards defined within you URI path. See Request routing for detailed information.

Example:

<?php

use \Can\Server\Request;
use \Can\Server\Route;

$route = new Route(
    '/hello/<name>',
    function (Request $request, array $args) {
        return 'Hello, ' . $args['name'] . '!';
    },
    Route::METHOD_GET
);

?>
